# 步骤编辑器界面设计文档

## 概述

本设计文档定义了 AutoFlow 自动化操作软件中步骤编辑器的界面设计。根据不同的步骤类型，提供定制化的配置界面，以便用户直观地配置每个步骤的参数。

## 设计原则

1. **类型驱动**：根据步骤类型动态显示不同的配置表单
2. **直观易用**：使用图标和分组来组织相关参数
3. **实时预览**：在配置过程中提供参数预览
4. **统一风格**：保持与主界面一致的视觉风格
5. **响应式布局**：适配不同屏幕尺寸

## 步骤类型分类

### 1. 鼠标操作类

#### 1.1 鼠标点击 (mouse_click)

**功能描述**：在指定位置或图片位置执行鼠标点击操作

**配置参数**：

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| 点击类型 | 下拉框 | 左键单击 | 左键单击/左键双击/右键单击/中键单击 |
| 点击位置 | 单选组 | 当前位置 | 当前位置/屏幕坐标/图片位置/相对坐标 |
| X坐标 | 数字输入 | 0 | 屏幕X坐标（点击位置为屏幕坐标时可用） |
| Y坐标 | 数字输入 | 0 | 屏幕Y坐标（点击位置为屏幕坐标时可用） |
| 图片路径 | 文件选择 | - | 目标图片路径（点击位置为图片位置时可用） |
| 相似度 | 滑块 | 0.9 | 图片匹配相似度（0.0-1.0） |
| 查找范围 | 下拉框 | 全屏 | 全屏/当前窗口/自定义区域 |
| 相对偏移X | 数字输入 | 0 | 相对图片中心的X偏移量 |
| 相对偏移Y | 数字输入 | 0 | 相对图片中心的Y偏移量 |
| 随机偏移 | 复选框 | 未勾选 | 是否在点击位置周围随机偏移 |
| 随机范围 | 数字输入 | 5 | 随机偏移范围（像素） |
| 延时(秒) | 数字输入 | 0 | 执行前等待时间 |

1. **动态显示**：根据选择的选项动态显示/隐藏相关配置项

## 实现建议

### 技术方案

1. 创建 `StepEditorDialog` 类，根据步骤类型动态构建表单
2. 使用 `QStackedWidget` 或动态添加/移除控件实现不同类型的界面切换
3. 为每种步骤类型创建对应的配置面板类
4. 使用信号槽机制处理控件交互和动态显示逻辑

### 文件结构

### 开发优先级

1. **第一阶段**：实现常用步骤类型（鼠标点击、键盘输入、等待、图片识别）
