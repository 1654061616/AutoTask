# AutoFlow 节点图编辑器界面设计文档

## 1. 设计概述

### 1.1 设计目标
将现有的步骤表格配置方式改为**节点拖拽式流程编排界面**，参考 Qflow 项目的节点图设计风格，提供更直观、可视化的流程编排体验。

### 1.2 界面风格
- **浅色主题**：白色网格背景，现代化视觉风格
- **节点颜色编码**：不同类型节点使用不同颜色标识
- **连线可视化**：清晰展示流程执行顺序和分支逻辑
- **拖拽交互**：支持节点拖拽移动、连接端口、删除等操作

### 3.2 节点结构

每个节点包含以下组件：

```
┌──────────────────────────────────┐
│  节点标题栏 (颜色背景 + 类型名)       │
│  ┌────────────────────────────┐  │
│  │ 图标 节点名称              │  │
│  └────────────────────────────┘  │
├──────────────────────────────────┤
│  参数配置区域                    │
│  ┌────────────────────────────┐  │
│  │ 参数标签: 输入框/下拉框     │  │
│  │ 参数标签: 输入框/下拉框     │  │
│  └────────────────────────────┘  │
├──────────────────────────────────┤
│  端口区域                        │
│  ● 输入端口                      │
│                                │
│  ● 输出端口                      │
│  ● 输出端口 (条件判断节点有多个)   │
└──────────────────────────────────┘
```

### 3.3 节点端口定义

| 节点类型 | 输入端口 | 输出端口 |
|---------|---------|---------|
| 开始节点 | 无 | 1个 |
| 结束节点 | 1个 | 无 |
| 普通操作节点 | 1个 | 1个 |
| 条件判断节点 | 1个 | 2个 (True/False) |
| 循环节点 | 1个 | 1个 (循环体) + 1个 (循环后) |

## 4. 交互设计

### 4.1 节点操作

| 操作 | 触发方式 | 行为 |
|------|---------|------|
| 创建节点 | 从工具栏拖拽节点到画布 | 在鼠标位置创建新节点 |
| 选中节点 | 单击节点 | 高亮显示节点，显示边框 |
| 拖拽移动 | 拖拽节点标题栏 | 移动节点位置 |
| 编辑节点 | 双击节点或选中后按Enter | 打开参数配置对话框 |
| 删除节点 | 选中后按Delete键 | 删除节点及其连线 |
| 复制节点 | 选中后按Ctrl+C | 复制节点配置 |
| 粘贴节点 | 按Ctrl+V | 在当前位置粘贴节点 |

### 4.2 连线操作

| 操作 | 触发方式 | 行为 |
|------|---------|------|
| 创建连线 | 拖拽输出端口到另一节点输入端口 | 创建节点间连接 |
| 删除连线 | 选中连线后按Delete键 | 删除连接线 |
| 断开连接 | 双击连线 | 断开连接 |

### 4.3 画布操作

| 操作 | 触发方式 | 行为 |
|------|---------|------|
| 平移画布 | 按住鼠标中键拖拽 | 移动画布视口 |
| 缩放画布 | 鼠标滚轮 | 缩放画布 |
| 框选多个节点 | 拖拽鼠标形成矩形 | 选中多个节点 |
| 取消选择 | 单击空白区域 | 取消所有选中 |

## 6. 连接线设计

### 6.1 连线样式

- **默认状态**：浅蓝色实线，2px宽度
- **选中状态**：深蓝色实线，3px宽度
- **激活状态**：绿色虚线，动画效果（执行中）

### 6.2 连线类型

| 类型 | 样式 | 用途 |
|------|------|------|
| 顺序连接 | 直线/贝塞尔曲线 | 节点顺序执行 |
| 条件分支 | 不同颜色曲线 | True/False 分支 |
| 循环连接 | 弧线 | 循环回跳 |

## 9. 实现要点

### 9.1 画布实现

- 使用 `QGraphicsScene` 作为画布场景
- 使用 `QGraphicsView` 作为视图，支持缩放和平移
- 网格背景使用 `QGraphicsGridLayout` 或自定义绘制

### 9.2 节点实现

- 继承 `QGraphicsWidget` 或 `QGraphicsItem`
- 自定义绘制节点外观（颜色、圆角、阴影）
- 实现端口拖拽连接功能

### 9.3 连线实现

- 使用 `QGraphicsPathItem` 绘制贝塞尔曲线
- 实现端口连接检测
- 支持连线选中和删除

### 9.4 数据持久化

- 节点图数据与现有步骤数据格式兼容
- 支持从旧格式（步骤表格）迁移到新格式（节点图）

## 11. 与现有系统集成

### 11.1 数据转换

- **步骤表格 → 节点图**：顺序转换，每个步骤对应一个节点
- **节点图 → 步骤表格**：按拓扑排序生成步骤列表

### 11.2 配置面板复用

- 节点参数配置复用现有的 `StepConfigPanel` 系列面板
- 通过 `StepEditorDialog` 编辑节点参数

### 11.3 执行引擎集成

- 节点图执行前转换为步骤列表
- 保持现有执行引擎不变
